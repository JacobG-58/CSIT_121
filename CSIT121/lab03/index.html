What I learned

  1. Tags are declared by the bracket symbols surrounding them.
  2. The format for a basic html page includes:
    (Doctype Declaration, Head, Title, Body, Headers, Paragraphs, and Closures Of All Tags Listed.)
  3. To close a tag, you must add a forward slash (/) before the its signifying letters.
    Example: "head" closes with "/head"

Lab 3

Published on 2/23/2025 by Jacob Geipe

I have spent some time learning about the basics of CSS and howw it can be effectively used to alter the styling of a web page, thus giving the user a more enjoyable experience. Similarly to HTML the syntax is composed of elements, that are used to assign various styling options to different assets. CSS elements have been easier to remember for me because their name is more descriptive and the parameters within each element are fairly straightforward on a surface level. However, it can become more tricky when determining what elements should be assigned to which text, and can get overwheleming if you want a higher level of stylish diversity. This is where selectors come in; id and class selectors make a world of difference when trying to diversify style in your webpage because they allow you to set styling rules that can be applied specifically to certain parts of your html file.

What I learned

  1. Class selectors must come after a "." id selctors must come after a "#".
  2. A style sheet is a file that is referenced directly by an html file, containing rules and parameters that define the styling choices for a web page.
  3. When creating borders, it is often a good idea to follow them up with a padding style change in order to ensure that the content within the borders is not cramped and messy looking.
